# ReturnShield

AI-assisted return fraud detection for Shopify stores.

## MVP Scope

- Shopify app shell
- Return risk review dashboard
- Customer fraud profile foundation
- Risk reason codes
- Shopify ingestion planned for Phase 2

## Commands

```bash
pnpm install
pnpm dev
pnpm typecheck
pnpm build
```

## Architecture Rules

- TypeScript only
- ESM only
- No default exports for app code
- React 19-ready
- React Router v8-ready
- Vite latest with Codespaces host patch
- Neo-Tech UI aesthetic
